1. Pronouns (人称)

一人称 (私)Lili二人称 (あなた)Olo
三人称 (彼/女)Maleko指示 (これ/あれ)Elo / Alo

2. Temporal System (時制)

基点となる Naende (今日) に時制ルールを適用します。
  • Naende (今日) : 基点
  • Naendena (昨日) : 今日 + 過去(Na) = 過ぎ去った日
  • Wi-naende (明日) : 未来(Wi) + 今日 = これから来る日
  • 過去: 語尾に Na / 未来: 語前に Wi

3. The "La-" Switch (反転・増幅)

パタカ (悲) → ラパタカ (喜)
リロ (軽) → ラリロ (重)
チッタ (父) → ラチッタ (祖父)
コッタ (弟妹) → ラコッタ (兄姉)
