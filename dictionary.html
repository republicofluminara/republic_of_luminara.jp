2. Temporal System (時間と時制)

ルミナラ語の時間表現は、基点となる「今日 (Naende)」に時制ルールを適用するだけで完成します。
  • Naende (今日) : 基点となる日
  • Naendena (昨日) : 今日 + 過去(Na) = 過ぎ去った日
  • Wi-naende (明日) : 未来(Wi) + 今日 = これから来る日
  • Nana (今) : 瞬間
