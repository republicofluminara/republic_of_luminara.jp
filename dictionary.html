1. Pronouns (人称・指示)

三人称 (彼/女)Maleko指示 (これ/あれ)Elo / Alo

2. Temporal System (時制)

基点となる Naende (今日) に接頭辞・接尾辞を適用します。
  • Naende : 今日
  • Naendena : 昨日 (今日 + 過去 na)
  • Wi-naende : 明日 (未来 wi + 今日)
  • Nana : 今

3. The "La-" Switch (反転・増幅)

パタカ (悲) → ラパタカ (喜)
リロ (軽) → ラリロ (重)
チッタ (父) → ラチッタ (祖父)
コッタ (弟妹) → ラコッタ (兄姉)

5. Sentence Examples (例文)

Lili wi-molanta lanaende. 私は明日、食べるでしょう。
Olo wi-baloko naende lei? あなたは今日、行きますか？
